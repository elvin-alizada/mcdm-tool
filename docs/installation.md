# Installation

## Stable release

To install Location Selection with MCDM, run this command in your terminal:

```sh
uv add mcdm_tool
```

Or if you prefer to use `pip`:

```sh
pip install mcdm_tool
```

## From source

The source files for Location Selection with MCDM can be downloaded from the [Github repo](https://github.com/elayy/mcdm).

You can either clone the public repository:

```sh
git clone git://github.com/elayy/mcdm_tool
```

Or download the [tarball](https://github.com/elayy/mcdm/tarball/master):

```sh
curl -OJL https://github.com/elayy/mcdm/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd mcdm_tool
uv pip install .
```
